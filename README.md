# 修仙模拟器

> 一款文字修仙 Roguelike 游戏，每一次抉择皆关机缘。

**在线体验** → https://katrina55553.github.io/Cult-Game/

## 简介

你是一名初入仙途的修士，从拜入宗门开始，经历修炼、奇遇、渡劫、抉择，最终走向属于你的结局。

每一次开局，你的灵根、出身、际遇都截然不同。是飞升成仙，还是堕魔陨落？是成为神仙眷侣，还是逍遥散修？一切取决于你的选择。

## 游戏内容

### 事件系统

131+ 个事件构成丰富的修仙世界，事件权重会根据你的行为动态调整：

- **主线剧情** — 拜入宗门、拜师问道、内门试炼、宗门大战、飞升抉择
- **情缘支线** — 路遇佳人、月下双修、道侣共渡天劫、眷侣证道
- **修炼体系** — 剑道、丹道、阵法、神识、血脉、灵兽、神兵、功法八大分支
- **秘境寻宝** — 大能遗迹、古战场、秘境阵眼，与苍穹阁天骄争夺宝物
- **宗门对战** — 天玄宗与苍穹阁的恩怨，从论道大会到灵脉之争
- **散修江湖** — 拒绝宗门后的独行之路，散修坊市、前辈指点、组建散修盟
- **墨离剧情** — 神秘孤僻的同辈魔修，秘境中的相遇、联手、背叛与复仇
- **负面事件** — 走火入魔、灵石失窃、妖潮来袭、心魔幻境，增加挑战
- **天象奇遇** — 血月之夜、九星连珠、古图残卷、阵法对决
- **出身专属** — 部分事件仅对特定出身触发（如「神秘信笺」仅限魔裔宿命、隐世遗脉）
- **路线专属** — 散修独占事件（灵脉枯竭、散修围剿等）、宗门独占事件

### 回溯系统

🔄 每局游戏有 1 次回溯机会，可回退到上一个选择前的状态。快照持久化保存，刷新页面不丢失。

| 血脉之主 | 血脉Tier≥3 + 金丹境 |
| 神兵至尊 | 神兵Tier≥3 + 金丹境 |
| 功法大成 | 功法Tier≥3 + 金丹境 |

### 灵宠系统

6 种灵宠可通过事件获取，最高可升至 3 阶：

- **妖狼** — 妖兽袭击中驯服
- **灵鹤** — 云中灵鹤奇遇
- **玄龟** — 玄潭老龟奇遇
- **赤焰虎** — 赤焰幼虎奇遇
- **雷鹰** — 雷鹰巢穴奇遇
- **冰凤** — 冰凤现身奇遇

### 修炼体系

八大修炼分支，各有独立等级和专属事件：

| 体系 | 等级 | 说明 |
|---|---|---|
| 丹道 | 未入门→丹徒→丹师→丹宗 | 炼丹、丹方、丹药 |
| 阵法 | 未入门→阵徒→阵师→阵宗 | 阵法参悟、布阵破阵 |
| 剑道 | 未入门→剑徒→剑师→剑圣 | 剑意领悟、剑阵试炼 |
| 功法 | 未入门→功徒→功师→功宗 | 功法参悟、万法归一 |
| 血脉 | 未觉醒→初觉→半血→纯血 | 血脉觉醒、血脉共鸣 |
| 神兵 | 无→凡器→灵器→仙器 | 铸炼神兵、重铸升级 |
| 神识 | 0-100 数值 | 影响灵兽驯养等 |
| 灵兽 | 1-3 阶 | 灵宠获取和培养 |

### 属性系统

- **根骨** — 影响修炼速度和突破成功率
- **悟性** — 影响功法领悟和丹道阵法
- **气运** — 影响所有概率型事件的成功率
- **因果** — 正值为善，负值为恶，影响特定事件和结局
- **心魔** — 越高越容易触发负面事件，达 80 则堕魔

## 界面系统

### 📊 属性面板
查看基础属性（根骨、悟性、气运、因果、心魔）、境界、寿元、血脉等详细信息。

### ⚔ 修炼面板
查看修炼路径、丹道/阵法/剑道/功法/神兵/神识等级、已习得功法和已铸神兵。

### 👜 乾坤袋
查看和管理所有拥有的物品：灵石、灵兽、功法、神兵、法宝、可使用物品。点击任意条目查看详情，可使用物品支持直接使用。

### 📜 剧情线
实时追踪已触发的剧情线进度，显示已完成步骤、当前步骤和下一步提示。

### 🔄 回溯
每局 1 次机会，回退到上一个选择前的状态。

## 技术栈

React 19 + TypeScript 6 + Vite 8 + Tailwind CSS 4

所有音效和背景音乐通过 Web Audio API 程序化生成，无外部音频文件。游戏进度保存在浏览器 localStorage 中。
